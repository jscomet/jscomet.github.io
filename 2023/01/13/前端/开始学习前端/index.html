在家摆了好久，趁着报名了抖音青训营，来开始学习前端了哈哈哈哈。浏览器上找到一个学习网站:MDM。

对前端有了一个小小的理解。

一 基础软件

文本编辑器： Visual Studio Code

网络浏览器：Firefox、Microsoft Edge、Chrome

二 网页开发基础：html、css、JavaScript。

有一个形象的比喻，html是做盒子，css是做衣服，JavaScript是做交互。这些是一个网站的基础构成。

  • html:

    超文本标记语言（英语：HyperText Markup Language，简称：HTML）是一种用来结构化 Web 网页及其内容的标记语言。网页内容可以是：一组段落、一个重点信息列表、也可以含有图片和数据表。

  • css:

    层叠样式表（Cascading Style Sheet，简称：CSS）是一门样式表语言，这也就是说人们可以用它来选择性地为 HTML 元素添加样式。

  • JavaScript:

    JavaScript（缩写：JS）是一门完备的动态编程语言。当应用于 HTML 文档时，可为网站提供动态交互特性。
